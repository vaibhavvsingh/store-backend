Store Backend API

This page contains the information about all the API routes this application provides and the expected input and output for the same.

Routes

Users Routes

Method Route Desc Query Body
POST /user/login Login - {username: String, password: String}
GET /user/logout Logout - -
POST /user/register Register - {username: String, password: String, Email: String, First Name: String, Last Name: String}
PATCH /user Update User - {user ID: Number, username: String, password: String, Email: String, First Name: String, Last Name: String}
DELETE /user Delete User - {userID: Number}

Product Routes

Method Route Desc Query Body
GET /product Get All Products - -
GET /product Get Paginated Data {page: Number} -
GET /product Search Paginated Data {page: Number, search: String} -
GET /product Get Individual Product {id: Number} -
POST /product Add New Product - {name: String, brand: String, desc: String, price: Number, sizes: JSON, img: String, adminToken: String}
PATCH /product Update Product - {id: String, name: String, brand: String, desc: String, price: Number, sizes: JSON, img: String, adminToken: String}
DELETE /product Delete Product - {id: Number, adminToken: String}
